🎬 1. Mon premier programme

Voici le programme le plus simple en Java. Il affiche un message dans la console :

public class MonPremierProgramme { public static void main(String[] args) { System.out.println("Bonjour le monde !"); } }
Bonjour le monde !

💡 Qu'est-ce que ça veut dire ?

public class → On crée une « classe » (un fichier de code Java).
public static void main → C'est la fonction principale, celle que Java exécute en premier.
System.out.println() → Affiche du texte dans la console.

⚠️ Attention ! En Java, le nom du fichier doit toujours correspondre au nom de la classe. Ici le fichier doit s'appeler MonPremierProgramme.java.

📦 2. Les variables

Une variable, c'est comme une boîte qui contient une information. On lui donne un nom et une valeur.

// Déclarer des variables int age = 14; // Un nombre entier double taille = 1.65; // Un nombre à virgule String prenom = "Adrien"; // Du texte boolean estEleve = true; // Vrai ou faux char initiale = 'A'; // Un seul caractère // Afficher les variables System.out.println("Je m'appelle " + prenom); System.out.println("J'ai " + age + " ans");
Je m'appelle Adrien
J'ai 14 ans
💡 Astuce : En Java, on utilise le + pour « coller » du texte et des variables ensemble. C'est ce qu'on appelle la concaténation.

🏷️ 3. Les types de données

En Java, chaque variable a un type qui dit quel genre de donnée elle contient :

Type Description Exemple
int Nombre entier 42
double Nombre à virgule 3.14
String Texte (chaîne de caractères) "Salut"
boolean Vrai ou faux true / false
char Un seul caractère 'A'

📌 Différence entre int et double

int stocke des nombres sans virgule (1, 42, -7).
double stocke des nombres avec virgule (3.14, 1.5, -0.001).

🔢 4. Les opérations

On peut faire des calculs comme en maths :

int a = 10; int b = 3; System.out.println(a + b); // Addition → 13 System.out.println(a - b); // Soustraction → 7 System.out.println(a * b); // Multiplication → 30 System.out.println(a / b); // Division → 3 System.out.println(a % b); // Modulo (reste) → 1
💡 Le modulo (%) donne le reste de la division. Par exemple : 10 ÷ 3 = 3 reste 1, donc 10 % 3 donne 1.

🔀 5. Les conditions (if / else)

Les conditions permettent de prendre des décisions dans ton programme :

int age = 16; if (age >= 18) { System.out.println("Tu es majeur !"); } else if (age >= 13) { System.out.println("Tu es adolescent !"); } else { System.out.println("Tu es un enfant !"); }
Tu es adolescent !

Voici les opérateurs de comparaison :

Opérateur Signification
== Égal à
!= Différent de
> Plus grand que
< Plus petit que
>= Plus grand ou égal
<= Plus petit ou égal

🔁 6. Les boucles

Les boucles permettent de répéter des instructions plusieurs fois.

🔄 La boucle for

Quand tu sais combien de fois tu veux répéter :

for (int i = 1; i <= 5; i++) { System.out.println("Tour numéro " + i); }
Tour numéro 1
Tour numéro 2
Tour numéro 3
Tour numéro 4
Tour numéro 5

💡 Comment lire une boucle for ?

int i = 1 → On commence à 1.
i <= 5 → On continue tant que i est ≤ 5.
i++ → À chaque tour, i augmente de 1.

🔄 La boucle while

Quand tu ne sais pas combien de fois ça va se répéter :

int compteur = 0; while (compteur < 3) { System.out.println("Compteur = " + compteur); compteur++; }
Compteur = 0
Compteur = 1
Compteur = 2

🧩 7. Les fonctions (méthodes)

Une fonction (on dit « méthode » en Java) est un bloc de code réutilisable. Au lieu de réécrire le même code, tu l'appelles par son nom !

📌 Fonction simple (sans retour)

public static void direBonjour() { System.out.println("Bonjour ! 👋"); } // Dans le main : direBonjour(); // Affiche : Bonjour ! 👋 direBonjour(); // Affiche : Bonjour ! 👋

📌 Fonction avec paramètres

public static void saluer(String nom) { System.out.println("Salut " + nom + " !"); } // Dans le main : saluer("Adrien"); // Affiche : Salut Adrien ! saluer("Théo"); // Affiche : Salut Théo !

📌 Fonction qui retourne une valeur

public static int additionner(int a, int b) { return a + b; } // Dans le main : int resultat = additionner(5, 3); System.out.println(resultat); // Affiche : 8

💡 void vs int / String

void = la fonction ne renvoie rien, elle fait juste une action.
int, String, etc. = la fonction renvoie un résultat qu'on peut stocker dans une variable.

📋 8. Les tableaux

Un tableau (array) permet de stocker plusieurs valeurs dans une seule variable :

// Créer un tableau de prénoms String[] prenoms = {"Adrien", "Théo", "Lucas", "Emma"}; // Accéder à un élément (⚠️ ça commence à 0 !) System.out.println(prenoms[0]); // Adrien System.out.println(prenoms[2]); // Lucas // Parcourir tout le tableau avec une boucle for (int i = 0; i < prenoms.length; i++) { System.out.println("Bonjour " + prenoms[i]); }
Adrien
Lucas
Bonjour Adrien
Bonjour Théo
Bonjour Lucas
Bonjour Emma
⚠️ Attention ! En Java (et dans presque tous les langages), les tableaux commencent à l'index 0, pas 1 ! Le premier élément est donc prenoms[0].

🔄 Boucle for-each (plus simple)

for (String p : prenoms) { System.out.println("Salut " + p + " !"); }
Salut Adrien !
Salut Théo !
Salut Lucas !
Salut Emma !

⌨️ 9. Lire une entrée utilisateur

Pour demander quelque chose à l'utilisateur, on utilise un Scanner :

import java.util.Scanner; public class Questionnaire { public static void main(String[] args) { Scanner clavier = new Scanner(System.in); System.out.print("Comment tu t'appelles ? "); String nom = clavier.nextLine(); System.out.print("Quel âge as-tu ? "); int age = clavier.nextInt(); System.out.println("Salut " + nom + ", tu as " + age + " ans !"); clavier.close(); } }
Comment tu t'appelles ? Adrien
Quel âge as-tu ? 14
Salut Adrien, tu as 14 ans !

💡 Les méthodes du Scanner

nextLine() → Lit du texte.
nextInt() → Lit un nombre entier.
nextDouble() → Lit un nombre à virgule.
close() → Ferme le scanner (bonne pratique !).
