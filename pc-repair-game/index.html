💡 Conseils

  • Utilisez les outils de diagnostic AVANT de remplacer un composant !
  • Le multimètre teste l'alimentation électrique.
  • Écouter les sons détecte les problèmes de disque dur ou ventilateur.
  • Vérifier les températures détecte les surchauffes.
  • Le BIOS affiche les erreurs et composants manquants.
  • Commencez par les solutions les moins chères !

📋 RÈGLES DU JEU

🎯 Objectif

Vous tenez un atelier de réparation PC. Des clients arrivent avec des ordinateurs en panne. Diagnostiquez la panne et cliquez sur le bon composant dans le PC pour le remplacer !

👥 Mode Équipe

  • Jouez à 2 : un Technicien (au clavier) et un Guide (avec le Manuel imprimé).
  • Le Technicien décrit les symptômes. Le Guide cherche dans le Manuel.
  • Ensemble, décidez quel composant remplacer dans le PC.

💰 Budget

  • Vous démarrez avec 500€.
  • Chaque composant a un coût (câble = 12€, CPU = 260€).
  • Bon composant → points + argent gagné 🎉
  • Mauvais composant → argent perdu, le PC ne marche toujours pas 😬
  • Budget à 0€ → game over !
