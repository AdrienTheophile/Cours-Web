🌐 Publie ton jeu Scratch sur une page web !

Apprends à créer ta propre page HTML pour montrer ton jeu

🏠 Accueil ← Retour aux projets débutants

C'est quoi le but ? 🤔

Tu as créé un super jeu sur Scratch ? Maintenant tu vas apprendre à le mettre sur ta propre page web en HTML !

Tu vas recopier un code HTML, y mettre le nom de ton jeu, ton prénom, et le lien de ton jeu Scratch. C'est parti ! 🚀

1 Récupérer le lien de ton jeu Scratch

Va sur scratch.mit.edu, ouvre ton projet et regarde l'URL dans la barre d'adresse.

Elle ressemble à ça : https://scratch.mit.edu/projects/123456789/

📝 Note bien le numéro de ton projet, tu en auras besoin !

2 Comprendre le code à recopier

Voici le code HTML complet. Tu dois le recopier dans un fichier .html sur ton ordinateur.

⚠️ Les 3 lignes en jaune sont celles que tu dois personnaliser !

📝 Le code à recopier

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mon Jeu Scratch</title>
    <style>
        body {
            font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
            background-color: #f0f2f5;
            color: #333;
            margin: 0; padding: 0;
            display: flex; flex-direction: column;
            align-items: center; justify-content: center;
            min-height: 100vh;
        }
        header { text-align: center; margin-bottom: 20px; }
        h1 { color: #ffaa00; font-size: 3em; margin: 0;
             text-shadow: 2px 2px 4px rgba(0,0,0,0.1); }
        .game-container {
            background-color: white; padding: 20px;
            border-radius: 15px;
            box-shadow: 0 10px 20px rgba(0,0,0,0.1);
            width: 485px; height: 402px;
            display: flex; justify-content: center;
            align-items: center;
        }
        iframe { border: none; width: 100%; height: 100%;
                border-radius: 8px; }
        footer { margin-top: 30px; text-align: center; }
        .credits { font-size: 1.5em; color: #555;
                  background-color: white; padding: 10px 20px;
                  border-radius: 25px;
                  box-shadow: 0 4px 6px rgba(0,0,0,0.05); }
        .credits strong { color: #4a90e2; }
    </style>
</head>
<body>

    <header>
        <h1>⭐ Mon Super Jeu ⭐</h1>  <!-- ✏️ CHANGE LE TITRE DE TON JEU ICI -->
    </header>

    <main class="game-container">
        <iframe src="https://scratch.mit.edu/projects/TON_NUMERO/embed"            <!-- ✏️ REMPLACE TON_NUMERO PAR LE NUMERO DE TON PROJET -->
                allowtransparency="true" width="485" height="402"
                frameborder="0" scrolling="no" allowfullscreen></iframe>
    </main>

    <footer>
        <p class="credits">Jeu fait par : <strong>Ton Prénom</strong></p>  <!-- ✏️ METS TON PRENOM ICI -->
    </footer>

</body>
</html>

3 Les 3 choses à personnaliser

Dans le code, tu dois changer 3 choses (les lignes en jaune) :

🎮 Ligne 1 : Remplace ⭐ Mon Super Jeu ⭐ par le nom de ton jeu

🔗 Ligne 2 : Remplace TON_NUMERO par le numéro de ton projet Scratch
    Exemple : si ton URL est scratch.mit.edu/projects/1191256461/, mets 1191256461

👤 Ligne 3 : Remplace Ton Prénom par ton vrai prénom
