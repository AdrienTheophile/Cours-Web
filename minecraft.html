📋 Les étapes pour créer ton pack

  1. Télécharge le pack vanilla — Clique sur le bouton ci-dessous pour récupérer le pack de textures de base de Minecraft 1.21.
  2. Décompresse le fichier ZIP — Fais un clic droit sur le fichier téléchargé → « Extraire tout ».
  3. Ouvre les textures avec Paint.NET — Va dans le dossier assets/minecraft/textures et ouvre les fichiers .png que tu veux modifier.
  4. Modifie les textures — Utilise Paint.NET pour dessiner tes propres textures pixel par pixel (16×16 pixels en général).
  5. Sauvegarde tes modifications — Enregistre chaque fichier en gardant le même nom et le même format .png.
  6. Re-compresse le dossier en ZIP — Sélectionne tous les fichiers du pack → clic droit → « Compresser en fichier ZIP ».
  7. Place le ZIP dans Minecraft — Va dans Options → Packs de ressources et ajoute ton pack !
